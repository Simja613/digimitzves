# DigiMitzves

DigiMitzves is an autonomous Shabbat and Yom Tov automation appliance.

The project is designed to help observant Jewish families automate household electrical devices while preserving reliability, predictability and complete offline operation.

Unlike cloud-based smart home systems, DigiMitzves is designed as a deterministic embedded appliance.

## Goals

- Fully autonomous operation
- No Internet dependency
- Deterministic scheduling
- Safe recovery after power loss
- Zigbee device support
- Raspberry Pi deployment
- Industrial software architecture

## Project Status

The project is currently under active development.

Current focus:

- Engine lifecycle
- Scheduler
- Job model
- Recovery
